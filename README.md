# Vue, ExpressJS 를 사용한 게시판 프로젝트

Vue3 문법에 익숙해지고 Javascript의 비동기적인 코드 구현을 연습하기 위한 프로젝트 입니다.

간단하게 CRUD 기능들이 구현되었고 추가로 포인트 제도와 같은 부가적인 기능들을 구현했습니다.

---
## 개발 환경
* Vue 3.2.13
* Pinia 2.1.7
* Javascript ES6
* ExpressJS 4.18.2
---

## 기술 스택
* Vue3
* composition API
* pinia
* pinia-plugin-persistedstate
* axios
* vue-router
* ExpressJS
* Multer
* MariaDB
* MySQL
